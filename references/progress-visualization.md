# Progress Visualization

Every project must maintain these dashboard files:

- `.autoresearch/dashboard/status.json`
- `.autoresearch/dashboard/progress.md`
- `.autoresearch/dashboard/timeline.ndjson`

Each update should expose:

- current phase
- current gate
- phase-review status
- loop counters
- current agent turn
- completion estimate
- active blockers
- active jobs
- backend and GPU status
- next orchestrator action
