# Scientific Method

This document provides frameworks and criteria for evaluating research methodology.

## Study Design Types

### Experimental Designs

| Design | Description | When to Use |
|--------|-------------|-------------|
| True Experiment | Random assignment to conditions | Establishing causation |
| Quasi-Experiment | Non-random assignment to conditions | When randomization impossible |
| Pre-Post Design | Measurements before and after intervention | Within-subject changes |
| Factorial Design | Multiple independent variables | Interaction effects |
| Crossover Design | Subjects receive all conditions in sequence | Within-subject comparisons |

### Observational Designs

| Design | Description | Strengths | Weaknesses |
|--------|-------------|-----------|------------|
| Cohort Study | Follow groups over time | Temporal sequence, multiple outcomes | Expensive, loss to follow-up |
| Case-Control Study | Compare cases to controls | Efficient for rare outcomes | Recall bias, selection bias |
| Cross-Sectional | Snapshot at one time point | Efficient, prevalence | No temporal sequence |
| Case Series | Multiple case descriptions | Hypothesis generation | No comparison group |

### Meta-Analytic Designs

| Design | Description | When to Use |
|--------|-------------|-------------|
| Systematic Review | Structured literature synthesis | Comprehensive evidence summary |
| Meta-Analysis | Statistical synthesis of studies | Quantitative effect estimation |
| Network Meta-Analysis | Compare multiple interventions | Indirect comparisons |

## Validity Framework

### Internal Validity

**Definition**: The degree to which observed effects can be attributed to the manipulated variables rather than other factors.

**Threats to Internal Validity**:

| Threat | Description | Mitigation |
|--------|-------------|------------|
| History | External events during study | Control group, short time frame |
| Maturation | Natural changes over time | Control group, appropriate time frame |
| Testing | Practice effects from repeated testing | Alternative forms, longer intervals |
| Instrumentation | Changes in measurement | Standardized procedures, calibration |
| Regression to Mean | Extreme scores tend toward average | Control group, multiple measurements |
| Selection | Pre-existing group differences | Random assignment, matching |
| Mortality | Differential dropout | Intent-to-treat analysis, tracking |
| Instrument Reactivity | Measurement changes behavior | Blind assessment, naturalistic observation |

**Assessment Questions**:

1. Were participants randomly assigned to conditions?
2. Were groups equivalent at baseline?
3. Was there differential attrition?
4. Were appropriate statistical controls used?
5. Were there any confounding variables?

### External Validity

**Definition**: The degree to which results can be generalized to other populations, settings, and times.

**Threats to External Validity**:

| Threat | Description | Mitigation |
|--------|-------------|------------|
| Population Validity | Sample not representative | Random sampling, diverse samples |
| Ecological Validity | Artificial setting | Naturalistic settings, field studies |
| Temporal Validity | Results time-bound | Replication across time |
| Interaction Effects | Selection-treatment interaction | Diverse samples |
| Reactive Arrangements | Hawthorne effect | Naturalistic observation, blind designs |

**Assessment Questions**:

1. How was the sample selected?
2. What population does the sample represent?
3. Is the setting natural or artificial?
4. Would results hold for other populations/settings?
5. When was the study conducted?

### Construct Validity

**Definition**: The degree to which measures capture the intended theoretical constructs.

**Threats to Construct Validity**:

| Threat | Description | Mitigation |
|--------|-------------|------------|
| Inadequate Preoperational Explication | Poor construct definition | Clear theoretical framework |
| Mono-operation Bias | Single measure/operation | Multiple measures/operations |
| Mono-method Bias | Single method | Multiple methods |
| Confounding Constructs | Measures capture multiple constructs | Purified measures, factor analysis |
| Method Variance | Shared method across measures | Different methods for different constructs |

**Assessment Questions**:

1. Are constructs clearly defined?
2. Are operationalizations appropriate?
3. Is there evidence for convergent validity?
4. Is there evidence for discriminant validity?
5. Do measures have established reliability?

## Control and Blinding

### Types of Control Groups

| Type | Description | When to Use |
|------|-------------|-------------|
| No-Treatment Control | No intervention | Baseline comparison |
| Placebo Control | Inactive treatment | Controlling expectancy effects |
| Active Control | Alternative treatment | Comparing effectiveness |
| Waitlist Control | Delayed treatment | Ethical alternative |
| Dose-Response Control | Different doses | Establishing optimal dose |

### Randomization Methods

| Method | Description | Considerations |
|--------|-------------|----------------|
| Simple Randomization | Random assignment | May create imbalance in small samples |
| Block Randomization | Balanced within blocks | Ensures balance across time |
| Stratified Randomization | Balance on prognostic factors | Requires knowing prognostic factors |
| Cluster Randomization | Groups randomized | Account for clustering in analysis |
| Minimization | Dynamic balancing | Complex but effective |

### Blinding Types

| Type | Who is Blinded | Prevents |
|------|----------------|----------|
| Open Label | No one | - |
| Single-Blind | Participants | Placebo effects, demand characteristics |
| Double-Blind | Participants and experimenters | Observer bias, differential treatment |
| Triple-Blind | Participants, experimenters, analysts | Analysis bias |

## Measurement Reliability

### Types of Reliability

| Type | Description | Coefficient |
|------|-------------|-------------|
| Test-Retest | Stability over time | Pearson r, ICC |
| Inter-Rater | Agreement between raters | Cohen's kappa, ICC |
| Internal Consistency | Items measure same construct | Cronbach's alpha, Omega |
| Parallel Forms | Equivalence of forms | Correlation between forms |

### Reliability Standards

| Reliability Type | Poor | Acceptable | Good | Excellent |
|------------------|------|------------|------|-----------|
| Test-Retest | < 0.50 | 0.50-0.70 | 0.70-0.85 | > 0.85 |
| Inter-Rater (ICC) | < 0.40 | 0.40-0.60 | 0.60-0.75 | > 0.75 |
| Inter-Rater (Kappa) | < 0.20 | 0.20-0.40 | 0.40-0.60 | > 0.60 |
| Internal Consistency | < 0.60 | 0.60-0.70 | 0.70-0.90 | > 0.90 |

## Evaluating Methodology Quality

### Quality Assessment Checklist

1. **Research Question**
   - [ ] Clear and specific
   - [ ] Testable hypothesis
   - [ ] Appropriate scope

2. **Study Design**
   - [ ] Appropriate for question
   - [ ] Control group if experimental
   - [ ] Appropriate comparison conditions

3. **Participants**
   - [ ] Sample size justified
   - [ ] Selection criteria clear
   - [ ] Demographics reported

4. **Measures**
   - [ ] Valid and reliable instruments
   - [ ] Appropriate for population
   - [ ] Clear operationalization

5. **Procedure**
   - [ ] Standardized protocol
   - [ ] Randomization documented
   - [ ] Blinding where appropriate

6. **Analysis**
   - [ ] Appropriate statistical tests
   - [ ] Assumptions checked
   - [ ] Missing data addressed

7. **Reporting**
   - [ ] Complete results reported
   - [ ] Effect sizes with confidence intervals
   - [ ] Limitations acknowledged

### Common Methodology Errors

1. **Design Errors**
   - Wrong design for the question
   - Inadequate control conditions
   - Missing randomization when needed
   - Inappropriate comparison groups

2. **Measurement Errors**
   - Using unvalidated measures
   - Poor operationalization
   - Single-item measures without justification
   - No reliability assessment

3. **Sampling Errors**
   - Convenience sampling without acknowledgment
   - Sample too small for effect detection
   - Non-representative sample
   - High attrition not addressed

4. **Analysis Errors**
   - Ignoring clustering in cluster designs
   - Wrong unit of analysis
   - Not accounting for repeated measures
   - Inappropriate use of parametric tests
